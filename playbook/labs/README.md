# Labs

Labs are practical exercises that turn the playbook into real build practice.

A path tells you what to learn. A lab tells you what to do.

## Available labs

| Lab | Best for | Link |
|---|---|---|
| First Real Project Lab | Fresh vibe coders building one real project from idea to handoff | [Open](./first-real-project-lab.md) |

## How labs work

Each lab includes:

- Objective
- Setup
- Step-by-step tasks
- Copy-ready prompts
- Checkpoints
- Final handoff
