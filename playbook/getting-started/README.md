## Why This Section Exists

Most people who start using coding agents make the same foundational mistakes:

1. They start building too early, before defining what the project truth is (Lesson 0)
2. They open an agent and immediately start describing what to build — without thinking about who is building it (Lesson 1 & 2)
3. They use multiple agents at once without any structure (Lesson 1 & 2)
4. They pick a stack based on habit or developer preference rather than product needs (Lesson 0 & 3)
5. They skip writing rules and spend time correcting the same behavior in every session (Lesson 4)

These five lessons address each of those directly.
