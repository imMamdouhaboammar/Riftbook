## Why This Section Exists

Most people who start using coding agents make the same four mistakes:

1. They open an agent and immediately start describing what to build — without thinking about who is building it
2. They use multiple agents at once without any structure
3. They skip writing rules and spend time correcting the same behavior in every session
4. They start building too early, before defining what done looks like

These four lessons address each of those directly.
