# Workflows

Step-by-step maps detailing sequential actions from project inception to completion. Workflows prevent context loss and make task execution repeatable.

## Workflows Index

- [Vibe Coding](./vibe-coding-workflow.md) - Rapid prototyping and development with AI-assisted IDEs.
- [Graph Intelligence](./graph-intelligence-workflow.md) - Use Graphify, GraphRAG, Graphiti, and Code Review Graph in the right order before risky AI-assisted work.
- [Debugging](./debugging-workflow.md) - Root-cause detection and error resolution workflows.
- [Research](./research-workflow.md) - Codebase analysis, mapping, and technical feasibility studies.
- [Design](./design-workflow.md) - Turning mocks and design tokens into functional components.
- [Shipping](./shipping-workflow.md) - Preparing and launching production builds.

## How to Add a Workflow

To add a new workflow:
1. Create a markdown file in this folder (e.g. [`workflows/testing-workflow.md`](./testing-workflow.md)).
2. Apply the format from [`templates/workflow-template.md`](../templates/workflow-template.md).
3. Add a link and description here in `workflows/README.md`.
