# Frameworks

Architecture-level projects and technical frameworks for building serious AI-assisted applications.

These are not small tools or prompt packs. They shape how a system retrieves, reasons, stores context, or works with data.

## Categories

- [RAG Frameworks](./rag/README.md) — Systems for Retrieval-Augmented Generation, knowledge retrieval, and private-data reasoning.
