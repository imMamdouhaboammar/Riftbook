## Operating rule

Riftbook should stay workflow-first.

Every new resource should help a builder do one of these better:

- Understand a project
- Configure an agent
- Improve output quality
- Review risky changes
- Debug faster
- Reduce context waste
- Build repeatable workflows
- Learn an important agentic concept
