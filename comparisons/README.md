# Comparisons

This section helps readers choose between tools that look similar but solve different problems.

Use comparisons when a builder might confuse two resources or use the wrong tool for the wrong workflow.

## Available comparisons

| Comparison | Use it when | Link |
|---|---|---|
| Graphify vs Code Review Graph | You need to choose between project mapping and review blast-radius analysis | [Open](./graphify-vs-code-review-graph.md) |
| GraphRAG vs Graphiti | You need to choose between document-corpus reasoning and temporal agent memory | [Open](./graphrag-vs-graphiti.md) |

## How comparisons should work

Each comparison should explain:

- What each resource is for
- Where they overlap
- Where they differ
- Which one to start with
- When to use both
- When neither is needed
- Example prompts or workflows
