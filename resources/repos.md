# Curated Repositories

External open-source codebases and libraries of high utility.

---

## Model Context Protocol (MCP) Servers
- **Link**: [Awesome MCP Servers](https://github.com/punkpeye/awesome-mcp-servers)
- **Topic**: Model Context Protocol
- **Evaluation**: A directory of community-built servers extending agent capabilities (PostgreSQL, Puppeteer, Github, etc.).
- **Best Takeaway**: Use existing community servers rather than writing custom integration scripts from scratch.

## RAG Infrastructure
- **Link**: [Microsoft GraphRAG](https://github.com/microsoft/graphrag) ([Internal Card](../frameworks/rag/graphrag.md))
- **Topic**: Graph-based Retrieval-Augmented Generation
- **Evaluation**: Modular graph-based pipeline to extract structured data from unstructured text using LLMs.
- **Best Takeaway**: Best for global/local queries on large complex datasets, but watch indexing costs closely.
